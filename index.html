Jquery.pagescroller

jQuery plugin for simple scrolling of large pages

jQuery PageScroller Plugin v0.1 documentation.

This PageScroller plugin will help you make scrolling large web pages less painful. It basically adds a custom clickable area to the left (or right) side of your page, which will scroll to the top or to the previous chapter of your page with just one click. It perfectly fits fixed layouts with long texts (e.g. books, articles, white papers, etc).

Usage.

First, download js and css and place them to the corresponding folders of your project. Then, add the following line to your html:

<script type="text/javascript" src="path/to/jquery.pagescroller.js"></script>
<link rel="stylesheet" type="text/css" src="path/to/jquery.pagescroller.css" />

To make everything work add the following lines to your javascript code:

$(function(){
    $.pagescroller.init(options) //options is an optional argument
})

Useful options

width (default: 100):

As expected, this will set the width of page scroller element.

position (default: 'left'):

This sets the position of the page scroller element. By default the element will appear at the left side of the page. Can be set either to 'left' or 'right'

triggerDistance (default: 100):

The distance between the cursor and corresponding page border which will trigger the page scroller show/hide.

showArrows (default: true):

Page scroller supports only scrolling upwards so far. If set to false, arrows won't be displayed within the scroller element.

bgColor (default: '#CFCFCF'):

The page scroller element background color

arrowColor (default: '#BFBFBF'):

Sets the color of arrows

textColor (default: '#BFBFBF'):

Sets the color of the text label within the page scroller

upText (default: 'To top'):

Sets the texts label displayed under the scroll-up arrow
